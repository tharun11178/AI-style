Purpose
=======

public/logo.png
  App logo shown in the browser and user interface.

public/favicon.ico
  Browser tab icon.

src/App.jsx
  Main React application and app routing/view logic.

src/App.js
  Compatibility export for App.jsx.

src/main.jsx
  React entry point.

src/index.css
  Main CSS entry file.

src/styles.css
  Full application styling.

src/components/
  Reusable UI components such as navbar, sidebar, loader, uploader,
  recommendation cards, and protected route wrapper.

src/pages/
  App screens for home, login, register, dashboard, photo upload,
  face analysis, recommendations, color palette, history, and profile.

src/services/
  API helper files that connect the frontend to the FastAPI backend.
