# Datasets

Place training or reference datasets for the AI Style Wellness Advisor here.

Suggested layout:

- `face_shapes/` for labeled face shape images or feature rows.
- `skin_tones/` for palette and undertone samples.
- `outfits/` for outfit metadata, occasion labels, and style tags.
- `hairstyles/` for hairstyle labels mapped to face shapes.

The app does not require these files to run. The admin ML assets endpoint lists anything added here so you can track what has been collected.
